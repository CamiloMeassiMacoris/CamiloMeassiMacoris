## Sobre mim

Sou um desenvolvedor focado em transformar ideias em aplicações web, úteis e bem estruturadas.

Gosto de aprender colocando a mão no código: criando interfaces, APIs, integrações, automações, projetos de estudo e pequenos produtos que resolvem problemas reais.

Atualmente meu foco está em:

- desenvolvimento web com JavaScript, React e Next.js;
- APIs e backends com Node.js, Express e bancos de dados;
- SQL, modelagem de dados e boas práticas de persistência;
- testes, arquitetura e organização de projetos;
- evolução contínua para atuar cada vez melhor em projetos.
